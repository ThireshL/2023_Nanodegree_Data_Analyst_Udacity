# 2023 Nanodegree Data Analyst - Udacity

This repository contains project work completed as part of the **Udacity Data Analyst Nanodegree** program in 2023. It serves as a portfolio demonstrating practical skills in data analysis, wrangling, investigation, statistical testing, and effective communication of findings.

## Projects Included

- **Weather Trends Report**  
  Analyzed weather datasets to identify trends and provide interpretations through exploratory data analysis and visualizations.

- **Investigate a Dataset**  
  Performed a detailed investigation on a dataset, assessing data quality, discovering patterns, and extracting insights.

- **A/B Testing**  
  Designed and analyzed A/B tests to inform decision-making, employing hypothesis testing and statistical analysis.

- **Data Wrangling**  
  Implemented techniques for cleaning and transforming messy datasets to prepare for accurate analysis.

- **Communicate Findings**  
  Created dashboards, reports, and presentations to effectively communicate data insights using visual and narrative techniques.
