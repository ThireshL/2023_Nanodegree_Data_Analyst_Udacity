# Nanodegree_Data_Analyst_Udacity
2023 Nanodegree Data Analyst - Udacity
This repository contains the project work completed as part of the Udacity Nanodegree program for Data Analysis in 2023. It serves as a portfolio of practical projects demonstrating key skills in data analysis, wrangling, investigation, and communication of findings.

Projects Included
Weather Trends Report
Analysis and report generation on weather trends using real-world datasets. Techniques include exploratory data analysis, visualization, and trend interpretation.

Investigate a Dataset
Comprehensive investigation of a dataset with detailed analysis, identification of data quality issues, patterns, and insights.

A/B Testing
Design and analysis of A/B tests to make data-driven decisions. Includes hypothesis formulation, statistical testing, and result interpretation.

Data Wrangling
Handling messy datasets with techniques for data cleaning, transformation, and preparation for analysis.

Communicate Findings
Development of effective data communication skills through dashboards, reports, and presentations with clear visuals and narrative.

Technologies and Tools
Python (Pandas, NumPy, Matplotlib, Seaborn)

Jupyter Notebooks

Statistical Analysis

Data Visualization

Data Wrangling Techniques

How to Use This Repository
Each project is organized in its own Jupyter Notebook file, with documented code and markdown explanations. To explore the projects:

Clone the repository

Open the Jupyter Notebooks in your preferred environment

Follow the step-by-step analysis and explanations within each notebook
